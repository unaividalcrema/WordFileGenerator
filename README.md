# WordFileGenerator
NodeJs and ReactJs app to generate word files

## Installation

Install node dependencies in backend folder

```bash
cd backend
npm install
```
Start backend express server
```bash
npm start
```

Install node dependencies in frontend folder

```bash
cd frontend
npm install
```
Start frontend react app
```bash
npm start
```
